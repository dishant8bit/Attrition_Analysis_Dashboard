📊 HR Analytics Dashboard (Power BI)

📌 Project Overview

This project focuses on analyzing employee data using Power BI to uncover workforce trends and support data-driven HR decisions. The dashboard provides insights into employee attrition, demographics, job satisfaction, salary distribution, and departmental performance. The goal of this project is to demonstrate business intelligence and data visualization skills using Power BI, making it suitable for GitHub portfolios, internships, and entry-level Data Analyst/Business Analyst roles.

🗂 Dataset Description

The dataset contains employee-related information used to analyze workforce performance and attrition patterns.

🔑 Key Columns

* EmployeeID – Unique employee identifier
* Age – Employee age
* Gender – Employee gender
* Department – Employee department
* JobRole – Employee job role
* Education – Education level
* MaritalStatus – Marital status
* MonthlyIncome – Monthly salary
* YearsAtCompany – Years worked at the company
* JobSatisfaction – Employee satisfaction rating
* WorkLifeBalance – Work-life balance rating
* Attrition – Whether the employee left the company (Yes/No)

🛠 Tools Used

* Power BI – Dashboard creation and visualization
* Power Query – Data cleaning and transformation
* DAX – Calculated measures and KPIs

🧹 Data Cleaning Steps

* Removed duplicate records
* Handled missing and inconsistent values
* Standardized column formats
* Verified data types for analysis
* Created calculated measures using DAX
